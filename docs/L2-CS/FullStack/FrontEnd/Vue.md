## Install and Config

### Install

- for production
  - `<script src="https://cdn.jsdelivr.net/npm/vue"></script>`
- by npm:
  - `npm install -g vue`
  - `npm uninstall -g create-react-ap`
- Vue CLI
  - `npm install -g @vue/cli`
  - `vue --version`

### Config Vue App

- `vue create MYAPP`
  - CLI to create project
- `vue ui`
  - go to GUI to create project
- `npm run server`
- `npm run build`

- Vue Devtools for Chrome extension
