### Flow Control

#### Condition Expression

- if-else if-else

  ```javascript
  if (a === "other value") {
    do_something;
  } else if (a === "another value") {
    do_something;
  } else {
    do_something;
  }
  ```

- switch-case

  ```javascript
  switch (x) {
    case 1:
      console.log("one");
      break;
    case 2:
      console.log("two");
      break;
    default:
      console.log("none");
  }
  ```

#### Loop Expression

- for

  ```javascript
  for (let i = 0; i < 100; i += 1) {
    do_something;
  }
  ```

- do-while

  ```javascript
  do {
    x += 1;
    console.log(x);
  } while (x < 100);
  ```

- while

  ```javascript
  while (x > 90) {
    x -= 1;
    console.log(x);
  }
  ```
